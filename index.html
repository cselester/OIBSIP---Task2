Study Center Management

For my college project, I developed a study center management website using HTML, CSS, JavaScript, PHP, and MySQL. This comprehensive website aimed to streamline the management of a study center by providing features such as student enrollment, course management, attendance tracking, and exam scheduling. Through HTML and CSS, I created an intuitive and visually appealing user interface. JavaScript was used to add interactivity and enhance user experience. PHP and MySQL were employed for server-side scripting and database management, respectively. This project showcased my proficiency in front-end and back-end web development, database integration, and problem-solving skills.
